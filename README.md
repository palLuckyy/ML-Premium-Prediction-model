# 🏥 Health Insurance Cost Predictor

A Machine Learning web application that predicts the **annual health insurance premium** based on user inputs such as age, income, BMI, smoking habits, and medical history.

🔗 **Live Demo**: https://ml-premium-predictuion-model.streamlit.app/

---

## 📌 Project Overview

This project focuses on building a complete **end-to-end ML pipeline**, from data preprocessing to deployment.

The model analyzes multiple real-world factors and provides an **instant premium estimate** through an interactive web interface.

---

## 🚀 Features

- 📊 Real-time insurance cost prediction  
- 🧠 Machine Learning model (XGBoost) for high accuracy  
- 🧹 Data cleaning and preprocessing  
- 📈 Exploratory Data Analysis (EDA)  
- 🎯 User-friendly UI built with Streamlit  
- 🌐 Fully deployed web application  

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, XGBoost  
- **Visualization:** Matplotlib / Seaborn (EDA)  
- **Deployment:** Streamlit  

---

## ⚙️ Input Parameters

The model considers the following inputs:

- Age  
- Gender  
- Marital Status  
- Number of Dependents  
- Annual Income (₹ Lakhs)  
- Employment Status  
- BMI Category  
- Smoking Status  
- Medical History  
- Region  
- Insurance Plan  
- Genetic Risk  

---

## 📈 Machine Learning Workflow

1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Model Training  
6. Model Selection (**XGBoost chosen for best performance**)
